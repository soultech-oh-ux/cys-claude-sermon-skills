> **본문**: 빌립보서 4:4-7
> **목표**: 30분 분량 주일 강해 설교 한 편

## 월요일 저녁 — 주제 잡기

### 입력
```
[당신]
이번 주일(11월 9일) 본문이 빌립보서 4:4-7이다.
한국 교회 청년·중년·노년이 골고루 섞인 회중. 200명 규모.
이번 본문으로 어떤 메시지를 전해야 할지 아직 막연하다.
```

### Claude 응답 (`sermon-topic-message-coach` 6단계)

**[1단계] 주제 명확화 — 질문**
> 이 본문에서 가장 마음에 와 닿는 단어 한두 개를 알려주세요. 또 이번 회중에게 응답하고 싶은 현 상황이 있다면?

```
[당신]
"기뻐하라"와 "평강". 요즘 청년 우울증 통계와 중년의 직장
스트레스가 마음에 걸린다.
```

**[2단계] 설교 목적 구체화**
> 이 설교 후 회중에게 어떤 변화가 일어나길 원하시나요?
> A) 인지적 변화 (생각이 바뀜)
> B) 감정적 변화 (마음이 움직임)
> C) 결단적 변화 (행동이 바뀜)
> D) 종합

```
[당신] D — 그러나 결단보다는 위로와 새로운 시각.
```

**[3단계] 핵심 메시지 5개 추천**
1. 환경에 좌우되지 않는 기쁨의 원천 — 임박하신 주
2. 기쁨은 명령이다 — 그러나 무리한 명령이 아니다
3. 염려를 기도로 — 평강이라는 약속의 통로
4. 옥중에서 나온 기쁨 — 우리 옥중도 기쁨의 자리가 될 수 있다
5. "주께서 가까우시니라" — 종말론적 임재 신학

**[4단계] 메시지 선택과 핵심 단어 추출**
```
[당신] 4번을 메인으로, 1번을 부주제로.
```
> ✓ 선택. 핵심 단어 3개 도출:
> - **임박(엥기스, ἐγγύς)** — 거리·시간의 양면
> - **기쁨(카라/카이로, χαρά/χαίρω)** — 명령형의 의미
> - **평강(에이레네, εἰρήνη)** — 사고를 지키는 군병

**[5단계] 함께 볼 5개 구절·5개 사건**
- 시 16:11, 사 26:3, 마 6:34, 살전 5:16-18, 약 4:8
- 사건: 시므온의 송영, 다윗의 시 32편, 바울의 빌립보 첫 방문, 사도들 채찍맞고 기뻐함, 한나의 기도

**[6단계] 7가지 작성법 중 추천**
> 본문 강해(2번) + 칼빈식 강해(7번)을 결합 권장. 본문 흐름을 따라가되 lectio continua 정신으로 5–7절을 한 호흡에.

### 산출물 (월요일 저녁 종료 시점)
```
주제: 주께서 가까이 — 환경을 넘는 기쁨과 평강
본문: 빌 4:4-7
핵심 단어: 임박, 기쁨, 평강
3대지 가설:
  1) 기뻐하라 — 명령 (4-5a)
  2) 주께서 가까우시니라 — 근거 (5b)
  3) 염려 → 기도 → 평강 — 결과 (6-7)
함께 볼 본문/사건 정해짐
```

## 화요일 오전 — 본문 다각도 분석

### 입력
```
[당신]
빌립보서 4:4-7 본문을 7가지 분석법으로 다각도 분석해줘.
가짜 해석 검증도 함께.
```

### Claude 응답 (`sermon-text-analysis-multimethod`)

**1) 원어 분석**
- v.4 χαίρετε ἐν κυρίῳ πάντοτε (카이레테 엔 큐리오 판토테)
  - 현재 명령형 2복: 지속적 명령
  - 주의: '판토테(항상)'는 부사. 시간이 아닌 모든 상황을 의미
- v.5 τὸ ἐπιεικὲς (토 에피에이케스, 관용·온유) — 자주 오역
- v.5 ἐγγύς (엥기스, 가까움) — 공간/시간 양의

**2) 구조 분석**
```
A  기뻐하라 (4)
  B  관용을 모든 사람에게 (5a)
    C  주께서 가까우시니라 (5b)  ← 중심
  B' 아무것도 염려하지 말고 (6a)
A' 평강이 너희 마음과 생각을 지키시리라 (7)
```

**3) 평행 본문**
- 빌 3:1 ("나의 형제들아 주 안에서 기뻐하라")
- 빌 1:25 (바울의 옥중 기쁨)
- 살전 5:16-18 (항상 기뻐하라)
- 시 32:11 (의인들아 여호와로 기뻐하라)

**4) 키아스무스 (위 구조와 동일, 5b가 중심)**

**5) 신학적 분석**
- 종말론적 기쁨 (재림 임박 의식)
- 기쁨 = 의무이자 은혜
- 평강 = 객관적 상태 (감정 아님)
- 군대 용어 φρουρήσει(프루레세이, 지킬 것이다) → 보초 서다

**6) 문학적 분석**
- 명령형 4개 연발 (기뻐하라 × 2, 보이라, 염려치 말라)
- 결과절(7절)이 약속으로 마무리
- 옥중서신 전체의 클라이맥스 부분

**7) 역사문화 배경**
- AD 60-62년 로마 1차 투옥 추정
- 빌립보 = 마케도니아 식민도시
- 회중 구성 다양성 (루디아·간수·노예)
- 박해 상황 (빌 1:29-30)

**[가짜 해석 검증]**
한국 강단에서 흔한 오해:
1. ❌ "기쁨 = 감정적 행복" → 카이레테는 결단의 기쁨
2. ❌ "기뻐하라 = 억지로라도 웃어라" → 칼빈도 거부한 해석
3. ❌ "주께서 가까우심 = 친밀감" → 1차 의미는 종말론적 임박
4. ❌ "에피에이케스 = 양보" → 정확히는 '권리의 절제'

## 수–목요일 — 설교문 초안

```
[당신]
이제 30분 분량 강해 설교문을 작성하자.
서론은 한국 청년 우울증 통계로 시작하고 싶다.

[Claude → sermon-emotive-writing-coach]

【서론 옵션 3가지】
A) 통계 기반 ─ 2024년 OECD 청년 우울증 1위 한국
B) 개인 이야기 ─ 옥중에서 시므온 노인의 송영
C) 역설 진입 ─ "감옥에서 쓴 기쁨의 편지"

[당신] A로 가자.

[Claude]
서론 작성:
"2024년 한국 청년의 자살률이 OECD 1위에 다시 올랐습니다.
20대 사망 원인 1위가 자살입니다. 우울증 환자 100만 명을 넘었습니다.
이 시대에 한 사람이 외칩니다. '기뻐하라.'
그것도 한 번이 아니라 두 번. '내가 다시 말하노니 기뻐하라.'
그런데 더 충격적인 사실은─ 이 사람이 지금 감옥에 있다는 것입니다.
자유로운 시민이 자유 없는 청년에게 '기뻐하라' 외친 것이 아닙니다.
사슬 찬 죄수가, 자유로워 보이는 우리에게 '기뻐하라'고 합니다.
빌립보서 4장 4절. 오늘 우리가 함께 들어갈 본문입니다."

→ 본론 1대지 작성 → 2대지 → 3대지 → 결론
→ 문장 5기술 / 문단 5기술 / 전체 5기술 점검
```
