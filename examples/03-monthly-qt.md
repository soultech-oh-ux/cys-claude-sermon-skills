# 시나리오 03 — 새벽기도 큐티 한 달치 만들기

**상황**: 4주간 새벽기도 본문으로 **요한복음 13–17장**(고별 설교)을 다루고 싶다.

---

## 1단계 — 한 달 본문 분배

```
[당신]
요한복음 13–17장을 한 달(30일) 새벽기도 큐티로 분배해줘.

[Claude]
W1 (1–7일): 13장 — 발 씻기심, 새 계명
W2 (8–14일): 14장 — 길과 진리, 보혜사 약속
W3 (15–21일): 15장 — 포도나무 비유, 세상의 미움
W4 (22–28일): 16장 — 보혜사의 사역, 슬픔이 기쁨으로
W5 (29–30일): 17장 — 대제사장적 기도

본문 분할 예시 (Day 1-7):
Day 1 — 요 13:1-11 (발 씻기심)
Day 2 — 요 13:12-20 (본을 보임)
Day 3 — 요 13:21-30 (배신자 가룟 유다)
Day 4 — 요 13:31-35 (새 계명)
Day 5 — 요 13:36-38 (베드로의 부인 예고)
...
```

## 2단계 — Day 1 큐티 자동 생성

```
[당신]
Day 1 (요 13:1-11)로 표준 분량 큐티 만들어줘.

[Claude → sermon-qt-original-text-based]
```

### 산출물 (예시)

```
═══════════════════════════════════════
Day 1. 사랑의 끝까지 — 발 씻기시는 주님
요한복음 13:1-11
═══════════════════════════════════════

[서론 — 예화]
2024년 한국에서 한 노숙인 목욕봉사 다큐멘터리가 화제였다.
17년간 매일 아침 노숙인의 발을 씻기고 양말을 갈아 신긴
목사님의 이야기였다. 카메라 앞에서 그는 이렇게 말했다.
"발을 씻기는 일은 사실 발을 씻기는 일이 아닙니다.
사람을 회복시키는 일입니다."
2000년 전 한 분이 이미 그것을 보여주셨다.

[원문 분석]
13:1 "사랑하시되 끝까지 사랑하시니라"
    헬라어: εἰς τέλος (에이스 텔로스)
    영어 음역: eis telos
    한글 발음: 에이스 텔-로스
    의미: 시간적 '끝까지' + 완전함의 '극도까지'
    → 두 의미가 동시에 작동

13:5 "씻기시고" νίπτω (닢토)
    부분적 씻음 (vs λούω 루오, 전신 목욕)
    → 영적 의미: 매일의 정결

[3대지 메시지]
1) 사랑은 한계가 있다 — 그러나 주님의 사랑엔 끝이 없다
   세상의 사랑은 조건적·계산적·소진적
   주님의 사랑은 무조건적·헌신적·영원적

2) 종이 되심의 역설 — 만유의 주가 발 앞에 무릎 꿇다
   당시 발 씻기는 가장 천한 노예의 일
   유대인은 동족 노예에게도 시키지 않았던 일
   창조주가 피조물 앞에 무릎 꿇으심

3) 너희도 행하라 — 본받음의 명령
   v.14-15 "내가 너희에게 행한 것 같이 너희도 행하라"
   - 가정에서: 배우자·자녀의 발(필요)을 씻기는 자세
   - 직장에서: 가장 낮은 자리의 동료를 섬김
   - 교회에서: 직분을 권력 아닌 섬김의 자리로

[적용 질문]
- 오늘 내가 발을 씻겨야 할 한 사람은 누구인가?
- 나는 어떤 마음으로 그 자리에 앉아 있는가?

[기도]
주님, 오늘 제가 가장 낮아져야 할 자리에서
가장 자존심을 내려놓아야 할 그 한 사람의 발 앞에
무릎 꿇을 용기를 주옵소서.
주께서 저에게 그렇게 하셨음을 기억하며.
예수님의 이름으로 기도드립니다. 아멘.
```

## 3단계 — 30일 일괄 생성 (배치 모드)

```
[당신]
Day 2부터 Day 30까지 동일한 형식으로 일괄 생성해줘.

[Claude]
한 번에 30일치 출력은 토큰 한도로 어렵습니다.
다음 두 가지 옵션을 제안드립니다:

[옵션 A] 7일씩 나눠 4번에 걸쳐 생성
[옵션 B] 매일 새 채팅에서 1일치씩 (가장 깊이 있음)

[당신] 옵션 B로 가자. 매일 새벽 5시에 다음 날치 만들겠다.
```

---

## 4단계 — 매일 한 편 생성 루틴

매일 새벽:
1. 그날 본문 1구절 입력
2. `sermon-qt-original-text-based` 자동 호출
3. 출력을 인쇄/PDF로 저장
4. 새벽기도 인도자에게 전달

**한 편당 생성 시간**: 2–3분

---

## 5단계 — 시리즈 일관성 점검

W1 종료 시점에:
```
[당신]
지난 7일 큐티를 일관성 측면에서 검토해줘.
같은 주제가 반복되거나 빠진 강조점이 있는지.

[Claude → sermon-text-analysis-multimethod의 통합 분석 모드]

【W1 7일 통합 검토】
- 반복 주제: "사랑"이 5번, "낮아짐"이 3번 (양호)
- 누락 강조: "성령"이 1번뿐 — 14장 진입 전 사전 도입 권장
- 균형: 적용 비중이 묵상보다 약간 큼 → W2에서 묵상 강화
```
