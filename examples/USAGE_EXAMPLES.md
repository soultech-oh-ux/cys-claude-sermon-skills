## 시나리오 1. 매주 주일 설교 한 편 준비 (가장 기본)

**상황**: 화요일 아침. 이번 주일 본문은 **빌립보서 4:4-7**. 막막하다.

### Step 1. 주제 잡기 (월요일 저녁)

```
[당신]
이번 주일 빌립보서 4:4-7로 설교하려는데, 어떤 주제로 가야 할지
아직 명확하지 않다.

[Claude → sermon-topic-message-coach 호출]
6단계 대화 시작:
1) 회중에게 어떤 변화가 일어나길 원하는지?
2) 한국 교회의 현 상황 중 이 본문이 응답하는 부분?
... (대화 진행)
→ 5개 추천 메시지 → 1개 선택 → 핵심 단어 추출
```

**산출물**: 핵심 메시지 1개, 핵심 단어 3개, 함께 볼 5개 구절

### Step 2. 본문 다각도 분석 (화요일 오전)

```
[당신]
빌립보서 4:4-7을 다각도로 분석해줘.

[Claude → sermon-text-analysis-multimethod 호출]
- 원문 분석: χαίρετε(카이레테, 기뻐하라) 현재 명령형의 의미
- 구조 분석: 4-7절 chiastic 구조
- 평행 본문: 빌 3:1, 살전 5:16, 시 32편
- 키아스무스: 기뻐하라 ↔ 평강이 지키리라
- 신학: 종말론적 기쁨, 임박한 재림(엥기스, 4:5)
- 문학: 명령 → 근거(주의 임박) → 권면 → 약속 구조
- 역사문화: 빌립보 옥중서신, AD 60-62년 추정
+ 가짜 해석 검증: "기쁨=감정"이라는 한국 강단 흔한 오해 식별
```

### Step 3. 핵심 단어 깊이 (화요일 오후)

```
[당신]
"기뻐하라(χαίρετε, 카이레테)" 단어 깊이 풀어줘.

[Claude → sermon-bible-dictionary 호출]
- 어원: χαίρω의 명령형 2인칭 복수 현재
- 70인역 용례: 시 32:11, 사 49:13
- 신약 용례: 마 5:12, 눅 10:20, 빌 4:4 (반복)
- 신학적 의미: 환경에 좌우되지 않는 종말론적 기쁨
- '행복(εὐδαιμονία)'과의 차이
- 한국어 번역의 한계: '기뻐하라'는 감정적 뉘앙스가 강함
```

### Step 4. 시대 배경 (화요일 저녁)

```
[당신]
바울이 옥중에서 빌립보 교회에 이 편지를 쓸 때 분위기는 어땠나?

[Claude → sermon-history-culture-geo-context 호출]
- AD 60-62년 로마 1차 투옥 (사도행전 28장 직후)
- 빌립보: 마케도니아 식민도시, 로마 시민권 보유 자부심
- 회중 구성: 루디아·간수·노예 등 다양한 계층
- 박해 상황: 빌 1:29-30 "고난도 은혜"
- 옥중 환경: 라이브러리 사슬에 묶임, 군인 24시간 감시
```

### Step 5. 칼빈 신학으로 깊이 (수요일 오전)

```
[당신]
이 본문에 대한 칼빈의 입장은 어떤가?

[Claude → sermon-calvin-institutes 또는 calvin-style-insight 호출]
- 칼빈 빌립보서 주석에서의 핵심 강조: "기쁨은 의무"
- 『기독교 강요』 III권 7장 섭리에 대한 신뢰
- "하나님께 가까이 계심" → 우리의 모든 염려를 맡길 근거
- 칼빈식 적용: 환경이 아닌 하나님의 통치를 보라
```

### Step 6. 설교문 작성 (수요일~목요일)

```
[당신]
이제 실제 설교문 초안을 잡아보자.
서론은 어떻게 시작하면 좋을까?

[Claude → sermon-emotive-writing-coach 호출]
서론 옵션 3가지:
A) 시사 예화 — 한국인의 우울증 통계 → 본문으로 진입
B) 개인 간증형 — 익명의 청중 한 명을 떠올리며
C) 역설 진입 — "감옥에서 쓴 기쁨의 편지"

→ 선택한 옵션으로 서론 작성 → 본론 3대지 → 결론
→ 문장·문단·전체 3계층에서 글쓰기 기술 적용
```

### Step 7. 회중 검증 (토요일 오후)

```
[당신]
완성한 설교문을 워킹맘과 미혼 청년 시각에서 들어줘.

[Claude → sermon-audience-feedback-persona 호출]
[페르소나 1: 워킹맘 김지영, 38세, 두 자녀]
"기뻐하라는 말이 처음엔 좀 부담스럽게 들렸어요.
이번 주만 해도 회사에서..." (구체적 반응)

[페르소나 2: 미혼 청년 박민수, 27세, 취준생]
"옥중에서 기뻐하라는 말은 멋있는데,
나는 지금 '주께서 가까우시다'를 못 느끼는데..." (의문)

→ 두 시각의 차이 → 설교 보강점 도출
```

**총 소요 시간**: 단계당 5-15분 = 약 1시간 ~ 1시간 30분
**산출물**: 깊이 있게 검증된 30분 분량 설교문 1편

## 시나리오 2. 4주 교리 설교 시리즈 기획

**상황**: 사순절 4주간 "그리스도의 십자가" 교리설교 시리즈를 기획하고 싶다.

```
[당신]
"그리스도의 대속" 주제로 4주 교리설교 시리즈 만들어줘.

[Claude → sermon-doctrinal-planner 호출]
4단계 구조 × 4주 = 총 4편

W1: 대속의 필요성 (왜 십자가인가)
   - 본문: 롬 3:23-26
   - 교리 진술: 죄와 진노의 현실
   - 변증/오해 교정: "사랑의 하나님이 왜 진노?"
   - 적용: 자기 의로움을 내려놓기

W2: 대속의 본질 (대신 죽으심)
   - 본문: 사 53:4-6, 벧전 2:24
   - 교리 진술: 형벌적 대속(Penal Substitution)
   - 변증: 도덕적 영향설·통치자설 비판
   - 적용: 죄책감에서 해방

W3: 대속의 효력 (한 번에 영원히)
   - 본문: 히 9:11-14
   - 교리 진술: 단번성(once-for-all)
   - 변증: 미사 반복의 신학적 문제
   - 적용: 매주 갱신되는 양심

W4: 대속의 적용 (믿음으로)
   - 본문: 갈 2:20
   - 교리 진술: 그리스도와의 연합
   - 변증: 이신칭의의 윤리적 함의
   - 적용: 십자가 아래의 삶

[당신]
W2를 루터의 시각에서 더 깊이 풀어줘.

[Claude → sermon-luther-coaching 호출]
- 루터의 "행복한 교환(fröhlicher Wechsel)"
- 『기독자의 자유』의 두 번째 명제
- 십자가 신학(theologia crucis) vs 영광의 신학
- 인용: "그가 우리의 죄를 자신의 것으로 삼으시고
  자신의 의를 우리의 것으로 주셨다"
```

**산출물**: 4주 시리즈 완성 기획안 + 신학적 깊이 + 매 주차 4단계

## 시나리오 3. 새벽기도 큐티 한 달치 만들기

**상황**: 요한복음 13~15장으로 한 달간 새벽 큐티 텍스트 필요.

```
[당신]
요한복음 13:1-17 본문으로 새벽 큐티 만들어줘.
원문 기반, 표준 분량으로.

[Claude → sermon-qt-original-text-based 호출]
[제목] 사랑의 끝까지 — 발 씻기시는 주님

[서론 — 예화]
2024년 한국에서 화제가 된 한 노숙인 목욕봉사 다큐멘터리...
(웹 검색으로 최신 자료 자동 보강)

[본문 — 원어 분석]
13:1 "끝까지(εἰς τέλος, 에이스 텔로스)"
   - 두 가지 의미: 시간적 끝 + 완전함의 끝
   - 한글 발음: 에이스 텔-로스
   - 영어 음역: eis telos

13:5 "씻기다(νίπτω, 닢토)"
   - 발만 씻는 부분 동작 (vs 루오, 전신 목욕)
   - 영적 의미: 일상의 죄책 정결

[3대지 메시지]
1) 사랑은 끝이 있다 — 그러나 주님의 사랑엔 끝이 없다
2) 종이 되심의 역설 — 만유의 주가 발 앞에 무릎 꿇다
3) 너희도 행하라 — 본받음의 명령 (3가지 적용)

[기도]
주님, 오늘 제가 가장 낮아져야 할 자리에서...
```

이를 30일치 반복하시면 한 달 새벽기도 자료가 완성됩니다.

## 시나리오 4. 본문비평 깊이 학습

**상황**: 막 16:9-20(긴 결말)이 정말 원본인지 사본 차이를 학술적으로 보고 싶다.

```
[당신]
마가복음 16:9-20을 사본별로 비교해줘. 원본인가 후대 추가인가?

[Claude → sermon-textual-criticism 호출]

【사본 증거 비교】
- 시내 사본(ℵ) — 16:8에서 종결, 9-20 없음
- 바티칸 사본(B) — 16:8에서 종결, 9-20 없음 (공백 있음)
- 알렉산드리아 사본(A) — 9-20 포함
- 베자 사본(D) — 9-20 포함
- 비잔틴 다수 사본 — 9-20 포함
- 시리아 페시타 — 9-20 포함
- 라틴 Vulgata — 9-20 포함

【원시 번역본 증거】
- Old Latin k 사본 — 짧은 결말
- 콥트어 사본 — 짧은 결말과 긴 결말 모두 존재

【본문비평 원칙 적용】
- 외적 증거: ℵ B의 무게 (가장 오래된 4세기)
- 내적 증거: 막 16:8 γάρ로 끝나는 것은 어색하지만,
  9-20절의 어휘가 마가복음 다른 부분과 다름
- 어려운 독법(lectio difficilior): 짧은 결말이 더 어려움
- 짧은 독법(lectio brevior): 짧은 결말 우선

【학자들의 평가】
- NA28: 9-20을 이중 대괄호로 표시 (후대 추가 가능성 높음)
- UBS5 등급: A (짧은 결말이 원본일 가능성 매우 높음)
- 메츠거(Metzger): 9-20은 2세기 추가
- 반대 입장: Burgon, Farmer (전통 본문 옹호)

【설교적 적용】
정통 교리는 손상되지 않으나, 설교 본문으로 사용 시
"이 부분은 후대 추가일 수 있으나..."로 단서 표시 권장.
```

## 시나리오 5. 회중 시각으로 설교 검증

**상황**: 토요일 저녁. 내일 설교문 완성. 8명 페르소나 전원 검증.

```
[당신]
[설교문 첨부]
이 설교를 8명 페르소나 전원이 듣고 솔직한 반응 줘.

[Claude → sermon-audience-feedback-persona 호출 — 전체 라운드]

1) 워킹맘 (김지영, 38세)
"솔직히 처음엔 또 '기뻐하라'야? 했어요. 월요일 출근길에..."

2) 미혼 청년 (박민수, 27세)
"옥중에서 기뻐하라는 게 멋있긴 한데,
나는 지금 직장도 없고 결혼도 멀고..."

3) 중년 사업가 (이성훈 안수집사, 52세)
"목사님 말씀이 옳지만, 이번 주에 거래처에서 어음 부도가..."

4) 은퇴 직전 권사 (정순자, 64세)
"60평생 신앙생활 한 사람한테 '기뻐하라' 새로울 게 없죠..."

5) 새신자 청년 (조유진, 24세)
"근데 '주의 임박'이라는 게 무슨 뜻이에요? 재림이라는데..."

6) 이혼 후 홀로 양육하는 어머니 (최은영, 40세)
"기뻐하라는 말이 너무 무거워요. 나는 그냥 견디는 것도..."

7) 고3 입시생 (강현우, 18세)
"수능까지 100일 남았는데 기뻐하라고 하시면..."

8) 1세대 홀몸 권사 (백영자 권사, 78세)
"60년 전 시집와서 영감 일찍 보내고 자식들 다 키워놓고..."

【종합 보강점】
- 청년·새신자가 '주의 임박' 의미 이해 못 함 → 부연 필요
- 워킹맘·이혼 어머니에 '기뻐하라'가 부담될 수 있음 → 경로 제시
- 입시생에 단기 미래 적용 부족 → 구체화 권장
```

## 시나리오 6. 연간(52주) 설교 계획 수립

```
[당신]
"믿음" 키워드로 52주 설교 계획 만들어줘.

[Claude → sermon-planner-52week 호출]

【1월: 새해 — 믿음의 출발】
W1 새해 첫 주 — 아브라함의 부르심 (창 12:1-4)
   핵심 구절 5개, 핵심 포인트 3개, 세부 주제 5개, 찬송가 5곡

W2 ... (52주 모두 자동 생성)

【부활절 시즌 자동 반영】
W14 종려주일 — 마 21:1-11
W15 부활절 — 요 20:1-18

【추수감사·성탄절 자동 반영】
W47 ... W52 ...

【한국교회 트렌드 반영】
W23 청년 부재 시대의 교회 — 행 2:42-47
W34 1인 가구 시대 — 전 4:9-12
W41 환경과 청지기 — 창 1:28
```

이후 매 주마다 시나리오 1을 반복하시면 됩니다.

## 시나리오 7. 신학자 코칭으로 깊이 더하기

**상황**: 같은 본문(롬 8:28)을 5명의 신학자 시각에서 비교하고 싶다.

```
[당신]
로마서 8:28 "모든 것이 합력하여 선을 이루느니라"를

【1. 어거스틴】
- 『하나님의 도성』 XXII권 24장
- 악의 결핍(privatio boni) 관점에서 해석
- 신정론(theodicy)의 핵심 본문
- 강조: 하나님은 악조차도 선의 도구로 사용하신다

【2. 루터】
- 『노예의지론』과 연결
- "감추어진 하나님(Deus absconditus)"
- 십자가 신학으로 풀이: 고난 속의 선
- 강조: 약속을 믿음으로 붙들라 (sola fide)

【3. 칼빈】
- 『기독교 강요』 I권 17장 "섭리의 적용"
- 예정과 섭리의 통합
- 선(τὸ ἀγαθόν)은 영적 선 = 그리스도의 형상 닮음
- 강조: 작정하신 하나님의 영광을 위함

【4. 바빙크】
- 『개혁교의학』 II권 창조론·섭리
- 일반 은총 관점 추가: 모든 자연 안의 보존
- 유기적 사고: 부분과 전체의 조화
- 강조: 은혜는 자연을 회복·완성한다

【5. 로이드 존스】
- 『로마서 강해』 8장 시리즈 21편
- "확실히 안다(οἴδαμεν)" — 인식론적 확신
- 실존적 진단: 고통 중 신자의 절망 vs 확신
- 강조: 부흥은 이 약속에 의지함에서 시작

【종합 비교표】
| 신학자 | "선"의 정의 | 핵심 강조 | 적용 방식 |
```
