# 시나리오 02 — 4주 교리 설교 시리즈 기획

**상황**: 사순절 4주간 "그리스도의 십자가" 교리설교 시리즈 기획

---

## 1단계 — 전체 4주 시리즈 기획안

```
[당신]
"그리스도의 대속" 주제로 사순절 4주 교리설교 시리즈 만들어줘.
대상: 일반 회중. 매주 30분 분량.

[Claude → sermon-doctrinal-planner]
```

### 4주 구조 (각 주 4단계: 본문 → 교리 진술 → 변증/오해 교정 → 적용)

| 주차 | 부제 | 본문 | 교리 진술 |
|---|---|---|---|
| W1 | 대속의 필요성 | 롬 3:23-26 | 죄와 진노의 현실 |
| W2 | 대속의 본질 | 사 53:4-6 / 벧전 2:24 | 형벌적 대속 |
| W3 | 대속의 효력 | 히 9:11-14 | 단번성(once-for-all) |
| W4 | 대속의 적용 | 갈 2:20 | 그리스도와의 연합 |

---

## 2단계 — 매 주차 신학 깊이 더하기

```
[당신]
W2 "형벌적 대속"을 루터 시각에서 깊이 풀어줘.

[Claude → sermon-luther-coaching]
```

### 산출물

- **루터의 "행복한 교환"** (fröhlicher Wechsel)
- 『기독자의 자유』 핵심 명제 인용
- 십자가 신학(theologia crucis) vs 영광의 신학
- 대속의 형벌적 측면을 부각하는 루터 설교 예시

```
[당신]
같은 주제를 칼빈은 어떻게 다루나?

[Claude → sermon-calvin-institutes]
```

→ 『기독교 강요』 II권 16-17장 인용·해설

---

## 3단계 — 변증/오해 교정 자료

```
[당신]
"형벌적 대속이 폭력적이다" "사랑의 하나님이 왜 진노?" 같은
현대의 비판에 어떻게 응답할까?

[Claude → sermon-doctrinal-planner의 변증 모듈]
```

### 산출물 — 응답 스크립트
1. **N. T. 라이트 등 대속 재정의 시도**의 핵심과 한계
2. **희생제사 본문(레 16장, 사 53장, 히 9장)**의 일관성
3. 한국 강단에서 흔한 회피("진노" 단어 자체 안 씀)의 위험
4. **사랑과 진노의 양립**: 칼빈식 응답

---

## 4단계 — 매 주차 설교문 작성

```
[당신]
W1 설교문 초안을 작성하자. 4단계 구조 따라.

[Claude → sermon-emotive-writing-coach + doctrinal-planner]
```

### W1 산출물 구조
```
[서론 — 5분]
사순절의 의미 + 회중을 십자가 앞으로 안내

[1단계: 본문 강해 — 7분]
롬 3:23-26 절별 분해
- v.23 "범죄하였으매" 부정과거 시제
- v.24 "값없이 의롭다 하심을 얻은 자 되었느니라"
- v.25 "화목제물(힐라스테리온)"

[2단계: 교리 진술 — 7분]
"우리는 모두 죄인이며, 하나님의 진노 아래 있다"
- 전적 부패 (개혁주의 입장)
- 진노는 인격적·법적 (vs 비인격적 자연법)

[3단계: 변증·오해 교정 — 7분]
- "나는 그렇게 큰 죄인은 아닌데?" — 율법의 영적 깊이
- "사랑의 하나님이 진노?" — 거룩하기에 진노

[4단계: 적용 — 4분]
- 자기 의로움을 내려놓기
- 죄책감 vs 회개의 차이
- 다음 주 W2에 대한 기대로 마무리
```

W2-W4도 동일한 패턴.

---

## 5단계 — 회중 검증

```
[당신]
W1 설교문을 8명 페르소나가 들으면 어떻게 반응할까?

[Claude → sermon-audience-feedback-persona]
```

→ 종합 보강점 도출, 특히:
- 새신자 청년: "진노"라는 단어 거부감 → 설명 보강
- 워킹맘: 일상과의 연결 부족 → 적용 강화
