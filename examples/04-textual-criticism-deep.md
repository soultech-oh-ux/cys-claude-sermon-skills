# 시나리오 04 — 본문비평 깊이 학습

**상황**: 박사과정 신학생이 **요한복음 7:53-8:11(간음한 여인 단락)**의 본문비평적 위치를 학술적으로 정리해야 한다.

---

## 1단계 — 본문비평 개관

```
[당신]
요한복음 7:53-8:11 단락을 사본별로 비교해줘.
원본인지 후대 추가인지 학술적으로 정리.

[Claude → sermon-textual-criticism]
```

### 산출물 — 사본 증거 종합

**[해당 단락이 없는 사본 — 증거 A]**
- P66 (200년경) — 없음
- P75 (3세기 초) — 없음
- 시내 사본 ℵ (4세기) — 없음
- 바티칸 사본 B (4세기) — 없음
- 워싱턴 사본 W (5세기) — 없음
- 시리아 페시타 — 없음
- 콥트어 사본(사히드, 보하이르) — 없음
- 아르메니아·조지아어 — 없음

**[해당 단락이 있는 사본 — 증거 B]**
- 베자 사본 D (5–6세기) — 있음 (현재 위치)
- 라틴 Vulgata — 있음
- 비잔틴 다수 사본 — 있음 (12세기 이후)
- 일부 사본은 누가복음 21:38 뒤에 위치
- 일부는 요한복음 7:36 뒤
- 일부는 요한복음 21:24 뒤

**[교부 증거]**
- Origen, Tertullian, Cyprian — 언급 없음
- 4세기 전반 교부 누구도 인용하지 않음
- 5세기 이후 점차 등장

## 2단계 — 본문비평 원칙 적용

```
[당신]
본문비평 원칙들을 차례로 적용해서 평가해줘.

[Claude → sermon-textual-criticism]
```

### 원칙별 평가

| 원칙 | 결론 |
|---|---|
| **외적 증거 (사본 무게)** | 짧은 본문(없음) 우세 |
| **내적 증거 (어휘·문체)** | 단락의 어휘가 요한복음과 다름. 누가복음 어휘에 가까움 |
| **어려운 독법(lectio difficilior)** | 짧은 본문이 더 어려움 (주께서 죄인을 정죄 안 함은 후대 강조 가능성) |
| **짧은 독법(lectio brevior)** | 짧은 본문 우선 |
| **사본 위치의 가변성** | 텍스트가 여러 위치를 떠돌았음 → 후대 삽입의 강한 증거 |

### 학자들의 평가
- **NA28**: 이중 대괄호 [[...]] 표시 (후대 추가 가능성 매우 높음)
- **UBS5**: A 등급 (확실하게 짧은 본문이 원본)
- **메츠거(Bruce Metzger)**: 진정한 사건 전승일 가능성, 그러나 요한복음 원본은 아님
- **D. A. 카슨**: 동일 입장
- **반대 입장**: Pickering, Hodges (Majority Text 옹호자) — 비잔틴 다수 본문이 원본

## 4단계 — 신학자들의 입장

```
[당신]
어거스틴, 칼빈, 루터, 바빙크는 이 단락을 어떻게 다뤘나?

[Claude → 4개 코칭 스킬 통합 호출]
```

### 신학자별 입장

**어거스틴 (354–430)**
- 자신의 시대 일부 사본에 없음을 인지
- 그러나 "신실한 자들의 두려움"으로 인해 삭제되었다고 추측
- (오늘 학자들은 어거스틴의 추측을 약한 변호로 봄)

**루터 (16세기)**
- 본문 사용함, 사본 비평 인지하지 못한 시대

**칼빈 (16세기)**
- 본문 인정, 그러나 주석에서 "요한 외 다른 사람의 손에서 왔을 가능성"을 시사

**바빙크 (19–20세기)**
- 사본 비평 결과 수용
- 정통 교리는 영향 받지 않음을 강조

**현대 복음주의 입장**
- D. A. 카슨, R. K. Wallace: 후대 추가, 그러나 *역사적으로 진정한 사건*일 가능성
- 정통 교리에 영향 없음

## 5단계 — 설교적 함의

```
[당신]
이 결론을 갖고 설교 본문으로 사용해도 되나? 어떻게 다뤄야 하나?

[Claude]

【설교 활용 가이드라인】

[옵션 A] 본문으로 사용 (주의 사항 있음)
- "이 단락의 사본 전승에는 논란이 있으나..."로 시작
- 핵심 메시지(긍휼·회개·정죄에 대한 신중함)는 다른 본문에서도 지지됨
- 회중 수준에 따라 사본 비평 설명을 생략 가능

[옵션 B] 본문 사용 자제
- 설교 본문은 더 견고한 본문 우선
- 이 단락의 메시지는 누가복음 7:36-50, 요한복음 4장 등에서 동일하게 발견

[옵션 C] 학술 강의에서 다루기
- 신학교 본문비평 강의 자료로
- 청년부·교사 훈련에서 사본 비평 입문 자료로

【주의 사항】
- 한국 강단에서 흔한 오해: "성경 어디에 있는지 다 같다"
- 정직한 설교자는 사본 차이를 인정하되 정통 교리의 안정성도 함께 보임
- 회중을 흔드는 도구가 아니라, 성경의 신뢰성을 더 견고히 하는 기회로
```

## 6단계 — 학술 정리

박사 논문이나 신학 저널 투고용 정리:

```markdown
# Pericope Adulterae (요 7:53-8:11) — 본문비평 종합

## I. 사본 증거
1. 부재 증거: P66, P75, ℵ, B, W, syp, copsa, copbo, geo
2. 존재 증거: D, byz (12세기 이후), it, vg
3. 위치 가변성: 누가복음·요한복음 사이를 떠돔

## II. 내적 증거
1. 어휘 분석: ἀναμάρτητος(8:7) — 요한복음 헤픽스
2. 문체: 누가복음에 가까움 (특히 8:6 "καταγράφω")
3. 신학: "정죄하지 않는 그리스도" 강조는 후대 강조

## III. 학자 평가 (Bruce M. Metzger, A Textual Commentary)
"본 단락은 명백히 4세기 이전 누구에게도 알려지지 않았다."

## IV. 정통 교리에의 영향
- 없음. 다른 본문에서 동일 메시지 지지됨
- 그리스도의 긍휼: 눅 7:36-50, 요 4장, 막 5장 등

## V. 결론
짧은 본문이 원본이며, 이 단락은 진정한 역사적 전승이
2–4세기 어느 시점에 요한복음에 삽입된 것으로 판단됨.
```
