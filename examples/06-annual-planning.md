# 사용 예시 06 — 1년 설교 연간 계획 수립

> **시나리오**: 12월 둘째 주, 새해를 앞두고 1년 52주 강단 설교 계획을 세워야 한다. 교회의 영적 흐름과 절기, 사회적 트렌드를 모두 반영한 신학적으로 균형 잡힌 연간 로드맵이 필요하다.

---

## 1. 상황 설정

- **대상 교회**: 출석 250명 규모의 도심 중형 교회
- **2025년 키워드(당회 합의)**: "회복(restoration)"
- **고려 사항**:
  - 절기(부활절·맥추감사절·추수감사절·성탄절) 자동 반영
  - 한국 교회의 시대적 트렌드(고립·세대갈등·번아웃·환경 위기) 부분 통합
  - 분기별 시리즈 구성(13주 × 4분기) 가능성
  - 각 주차에 활용할 본문 후보·핵심 메시지·찬송가까지 사전 수립
- **기간**: 2주 집중 작업

---

## 2. 활용 skill 매핑

| 단계 | 사용 skill | 역할 |
|---|---|---|
| 키워드 → 52주 골격 | `sermon-planner-52week` | 키워드 1개로 연간 구조 자동 생성 |
| 분기별 시리즈 보강 | `sermon-doctrinal-planner` | 교리 시리즈 한 분기 삽입 |
| 사회 컨텍스트 | `sermon-topic-research-multidisciplinary` | 시대 흐름과 회중 현실 |
| 절기 본문 깊이 | `sermon-history-culture-geo-context` | 부활절·성탄 본문 배경 |
| 신학적 균형 점검 | `sermon-calvin-institutes` | 4권 골격으로 1년 신학 분포 검증 |

---

## 3. Day 1 — 1년 골격 자동 생성

### 3.1 첫 호출 — 52주 자동 생성

**입력**:
> "2025년 연간 설교 키워드는 '회복'이다. 52주 설교 계획을 만들어줘. 출석 250명 도심 교회이고, 4대 절기는 모두 반영해줘."

**호출되는 skill**: `sermon-planner-52week`

**산출물 (요약 발췌)**:

```markdown
# 2025년 52주 설교 계획 — 키워드 "회복(Restoration)"

## 1분기(1-13주): 자아의 회복 — 하나님 형상의 회복
- W1 (1/5): 새해 첫 주일 — "회복의 시작은 듣는 것"  (사 30:15)
- W2: 깨어진 형상, 회복되는 형상 (창 1:26-27)
- W3: 죄와 자아의 분열 (롬 7:14-25)
- W4: 그리스도 안에서의 새 자아 (고후 5:17)
... (분기별 13주 모두 본문/핵심 포인트/찬송가 포함)

## 2분기(14-26주): 관계의 회복 — 가정·이웃·세대
- W14 (4/6): 사순절 4주차 — "용서의 사슬을 끊다" (마 18:21-35)
- W17 (4/20): **부활주일** — "관계 회복의 첫 열매, 부활" (요 21:15-19)
... 

## 3분기(27-39주): 교회 공동체의 회복
- W31 (8/3): 맥추감사절 — "잊혀진 추수의 노래" (신 16:9-12)
- ...

## 4분기(40-52주): 세상의 회복 — 환경·사회·종말
- W47 (11/23): 추수감사절 — "감사가 회복을 부른다"
- W51 (12/21): **성탄절 직전 주** — "회복의 약속, 임마누엘"
- W52 (12/28): 송년주일 — "회복의 길 위에서"
```

각 주차마다 **본문 5개 후보 + 핵심 포인트 3개 + 세부 주제 5개 + 찬송가 5곡**이 자동 생성됩니다.

> **포인트**: 첫 호출만으로 1년 골격의 80%가 잡혔습니다. 다음 단계는 보강·검증입니다.

## 4. Day 3 — 분기별 균형 점검 (신학적 분포)

### 4.1 칼빈 4권 구조로 1년 분포 검증

**입력**:
> "위에서 만든 52주 계획을 칼빈 『기독교 강요』 4권 구조에 비추어 분석해줘. 신학적 균형이 잡혀 있는가? 어느 영역이 과대/과소 대표되고 있는가?"

**호출되는 skill**: `sermon-calvin-institutes`

**산출물 (요약)**:

| 강요 권 | 주제 | 1년 중 비중 | 평가 |
|---|---|---|---|
| 1권 — 창조주 인식 | 8주 (15%) | 정상 |  |
| 2권 — 그리스도와 구속 | 14주 (27%) | **과대 대표** — 부활절기 집중 |  |
| 3권 — 성령·연합·칭의 | 12주 (23%) | 정상 |  |
| 4권 — 교회·성례·국가 | 6주 (12%) | **과소 대표** ⚠ |  |
| 절기·시사 | 12주 (23%) | — |  |

**Claude의 권고**:
- 4권(교회론·성례·세상 속 교회) 비중이 낮음 → 3분기 "교회 공동체 회복" 시리즈에 성례·세례·치리·국가 관계 주제 1-2주 보강
- 2권 비중은 부활절·성탄절 때문에 자연 증가 → 적절

### 4.2 W30-W34 보강 호출

**입력**:
> "3분기 27-39주 '교회 공동체의 회복' 시리즈에 성례(세례·성찬), 권징, 교회와 국가 관계를 다루는 3주를 추가하고 싶다. 어디에 끼워 넣어야 흐름이 자연스러운가?"

**산출물**: 흐름 분석과 함께 W30·W33·W36에 3개 주제 삽입 권고. 각각 본문·핵심 메시지 포함.

## 5. Day 5 — 한 분기를 교리설교 시리즈로

### 5.1 1분기를 "회복되는 자아 — 사도신경 강해" 시리즈로 변환

**입력**:
> "1분기 13주를 사도신경 강해 시리즈로 만들고 싶다. '회복' 키워드와 연결된 교리설교 기획안을 만들어줘."

**호출되는 skill**: `sermon-doctrinal-planner`

**산출물 (개요)**:

```markdown
# 1분기 교리설교 시리즈 — "회복되는 신앙고백: 사도신경"

W1: "나는 믿습니다 — 회복의 첫 고백" (롬 10:9-10)
W2: "전능하사 천지를 만드신 — 창조 신앙의 회복" (창 1:1)
W3: "성부 하나님 — 아버지 하나님의 회복" (눅 15:11-32)
W4: "예수 그리스도 — 주(主)의 주권 회복" (빌 2:9-11)
W5: "동정녀 마리아에게 나시고 — 성육신의 신비" (눅 1:26-38)
W6: "본디오 빌라도에게 고난을 받으사 — 십자가의 능력" (마 27:11-26)
W7: "장사된 지 사흘 만에 부활 — 부활 신앙의 회복" (고전 15:1-11)
W8: "하늘에 오르사 — 승귀하신 그리스도" (행 1:9-11)
W9: "성령을 믿사오며 — 성령론의 회복" (요 14:16-17)
W10: "거룩한 공교회 — 교회론의 회복" (엡 4:1-6)
W11: "성도가 서로 교통하는 것과 — 교제의 회복" (행 2:42-47)
W12: "죄를 사하여 주시는 것과 — 용서의 회복" (요일 1:8-10)
W13: "몸이 다시 사는 것과 영원히 사는 것 — 종말 신앙" (계 21:1-7)
```

각 주차마다 **본문 → 교리 진술 → 변증/오해 교정 → 적용** 4단계 구조가 함께 산출됩니다.

> **포인트**: 키워드("회복") + 교리 시리즈(사도신경) 결합으로 신학적 깊이와 일관성이 동시에 확보됩니다.

## 6. Day 8 — 회중 현실과 시대 흐름 통합

### 6.1 사회 컨텍스트 보강

**입력**:
> "회중 250명 도심 교회. 30-50대 직장인 가정 비율이 높다. 2025년 한국 사회의 '회복' 주제와 맞닿은 현실 이슈 5가지만 다학제로 정리해줘. 1년 설교에 깔개로 쓰겠다."

**호출되는 skill**: `sermon-topic-research-multidisciplinary`

**산출물**: 11개 분야 중 5개를 추출하여 — 번아웃(심리)·관계 단절(사회)·생성형 AI 노동 충격(기술/산업)·기후 불안(환경)·세대 분열(정치) 각각에 통계·연구·예화·적용 포인트를 정리.

### 6.2 매핑

이 5개 이슈를 52주 계획 위에 분기별로 1-2개씩 분배하여, 한 주차 한 주차의 "오늘의 적용 포인트"가 회중 현실과 직접 연결되도록 만듭니다.

## 7. Day 11 — 절기 4대 본문 배경 깊이 다지기

### 7.1 부활주일·성탄절·맥추·추수 4대 절기 본문 미리 깊이 분석

**입력**:
> "W17 부활주일 본문이 요한복음 21:15-19다. 1세기 갈릴리 어부 문화·생선 굽는 자리·세 번 묻는 의미를 역사·문화·지리적으로 풀어줘. 설교 깊이를 다지는 용도다."

**호출되는 skill**: `sermon-history-culture-geo-context`

**산출물**: 디베랴 호숫가의 새벽 어업 관습, 153마리 큰 물고기의 어종 추정, 숯불(ἀνθρακιά)이 베드로의 부인 장면(요 18:18)과 같은 단어라는 점, 1세기 어부 공동체의 사회 구조 등 — 부활주일 강단을 위한 풍부한 배경 자료.

부활주일·성탄절·맥추·추수 4대 절기 본문을 4일에 걸쳐 모두 사전 처리합니다.

## 8. Day 14 — 최종 마스터 플랜 정리

### 8.1 구조

작업 후 손에 남는 것:

```
2025_연간설교계획_회복.md
├── 0. 연간 비전 선언문
├── 1. 1분기 — 사도신경 강해 13주 (교리 시리즈)
├── 2. 2분기 — 관계 회복 13주 (사순절·부활절·성령강림 포함)
├── 3. 3분기 — 교회 공동체 회복 13주 (성례 3주 보강)
├── 4. 4분기 — 세상의 회복 13주 (감사·대림·성탄 포함)
├── 부록 A — 4대 절기 본문 배경 자료
├── 부록 B — 회중 현실 5대 이슈 매핑표
└── 부록 C — 칼빈 4권 분포 검증 결과
```

### 8.2 운용 방식

- 매주 설교 준비 때 이 계획서를 열어 해당 주차를 확인
- 그 주의 본문에 따라 추가 skill(예: `sermon-text-analysis-multimethod`, `sermon-emotive-writing-coach`) 호출하여 살을 붙임
- 분기 종료 때마다 회중 반응을 보고 다음 분기를 미세 조정

## 10. 응용 제안

- **3년 시리즈**: 키워드 3개("회복" → "동행" → "보냄")로 3년 연속 계획
- **부교역자 분담**: 분기별로 1명씩 책임지고 동일 skill 셋으로 표준화
- **월간 리뷰**: 매월 마지막 월요일 당회에서 다음 달 4주를 점검·확정

> **결론**: `sermon-planner-52week` 한 번의 호출이 출발점을 잡아주고, 다른 7개 skill이 신학·역사·시대·교리·문체의 살을 붙입니다. 1년 설교 계획은 더 이상 "감"이 아니라 **검증된 시스템**의 산출물이 됩니다.
