## 4가지 호출 패턴

### 패턴 A — 단일 페르소나
```
[당신]
[설교문 첨부]
이 설교를 워킹맘 시각으로만 들어줘.

[Claude → 김지영 페르소나 단독 응답]
"음... 솔직히 처음 5분은 머릿속이 복잡했어요.
어제 둘째가 열이 나서 새벽에 두 번 깼거든요.
'기뻐하라'는 첫 마디가 솔직히 좀 부담스럽게 들렸어요.
'또 무리한 명령이구나' 했는데..."
```

### 패턴 B — 다중 비교
```
[당신]
워킹맘·청년·새신자 3명만 비교해서 듣게 해줘.

[Claude → 3명 시각 + 차이점 분석]
```

### 패턴 C — 추천형
```
[당신]
이 설교가 누구에게 가장 강하게 들리고, 누구에게 약할까?

[Claude]
가장 강하게 들리는 페르소나:
  - 정순자 권사 — 평생 신앙 다져진 분에게 깊이 있음
가장 약하게 들리는 페르소나:
  - 조유진 (새신자) — 종말론 용어가 생소함
  - 강현우 (고3) — 단기 미래 적용 부족
```

### 패턴 D — 전체 라운드
```
[당신]
8명 모두에게 짧게 들어줘.

[Claude → 8명 각자 짧은 반응 + 종합 보강점]
```

## 실제 응답 예시 — 4명 추출

### 김지영 (워킹맘, 38세)
> 솔직히 '기뻐하라'는 첫 마디가 부담스러웠어요. 이번 주만 해도 회사에서 야근 두 번, 어제는 둘째가 열이 나서 새벽에 두 번 깼고요. '기뻐하라'고 하시는데 '또 무리한 명령이구나'… 그런데 목사님이 '주께서 가까이' 부분에서 '환경이 아니라 임재'라고 하실 때 살짝 호흡이 풀렸어요. '그래, 내 환경이 좋아져서 기뻐하는 게 아니구나' 했어요. 다만 '관용을 모든 사람에게'에서 시댁 얘기가 떠올라서… (잠시 한숨)

### 박민수 (미혼 청년, 27세)
> 옥중에서 쓴 편지라는 도입이 좋았어요. 멋있다 생각했어요. 그런데 듣다가 갑자기 마음이 무거워졌어요. 바울은 옥중에서도 사명이 있었잖아요. 저는 직장도 없고, 사명은커녕 다음 달 카드값이 걱정인데. '주께서 가까우시니라'가 솔직히 안 와닿았어요. 가까우시면 왜 제 면접 결과가 다 떨어지나… 이런 마음이 죄송하지만 솔직한 마음이에요.

### 이성훈 (중년 사업가 안수집사, 52세)
> 목사님 말씀이 옳습니다. 그런데 제가 이번 주에 거래처에서 어음 부도가 두 건 들어와서… 30년 신앙생활 했지만, '아무것도 염려하지 말라' 부분에서 솔직히 '쉽게 말씀하시네요' 했어요. 그래도 '구하는 것을 감사함으로 아뢰라' 부분에서 마음이 움직였어요. 감사함 없이 청구서만 들고 갔던 제 기도가 떠올랐어요.

### 백영자 (1세대 홀몸 권사, 78세)
> 60년 신앙생활 했어. 영감 일찍 보내고 자식들 다 키워놓고 이제는 새벽기도 가는 거 하나가 일이야. 목사님이 '주께서 가까우시니라' 하실 때 가슴이 뜨거워졌어. 가까우시지. 곧 나도 그 분 앞에 갈 텐데. 청년들에겐 '재림'이지만 나에겐 '오늘 밤이라도'야. 평강이라는 단어가 좋았어. 평강.

## 종합 보강점 자동 도출

```
[Claude]

【반복적으로 약한 영역】
1. 새신자에게 "주의 임박" 의미가 막연 → 짧은 부연 권장
2. 청년·이혼 모에게 '기뻐하라'가 부담 가능 → 경로 제시 필요
3. 고3에게 단기 미래 적용 부족 → 학업·진로 한 줄 추가

【의외로 강한 영역】
1. 1세대 권사에게 '주의 가까우심'이 종말론적으로 깊이 들림
2. 사업가 안수집사에게 '감사함으로 아뢰라'가 강하게 작용
3. 워킹맘에게 '환경이 아닌 임재' 부분이 접점

【보강 권고】
- 서론 후반에 "오늘 이 자리 누군가는…" 식으로
  3가지 처지(취업/사업/노후)를 짧게 호명
- '주께서 가까우시니라'를 두 가지 의미로 해설:
  ① 청년·새신자에게 — 재림의 임박
  ② 노년에게 — 부르심의 임박
- 세 번째 대지 마지막에 입시생을 위한 한 줄 적용 추가
```

## 페르소나 검증의 한계

```
[당신]
이 페르소나 검증을 100% 신뢰해도 되나?

[Claude]

【한계 인식】
1. 8명은 한국 교회의 *모든* 결을 대표하지 않음
2. AI 시뮬레이션은 *가능성 있는* 반응이지 실제 회중 그 자체는 아님
3. 회중 개인의 깊은 영적 상태는 시뮬레이션 불가
4. 페르소나는 일반화된 패턴, 본인 회중과 차이 가능

【권장 활용】
- 검증의 *시작점*으로 활용
- 본인 회중을 떠올리며 페르소나 미세 조정
- 실제 회중 한두 명과의 사전 대화로 보완
- 강단 후 솔직한 피드백 통로 마련
```
